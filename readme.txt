Simple Lazy loader for lazy loading images.

== Description ==
IC Lazy Load is a very lightweight plugin for lazy loading images. It will reduce your website load times. Image will not load untill it's visible in the viewport. It's work for post images, post thumbnail, gravatar images and text widget images.

== Installation ==
1. Download and unzip plugin
2. Upload the 'ic-lazy-load' folder to the '/wp-content/plugins/' directory,
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How to configure this plugin? =
You just have to install and activate this plugin, that's it!
